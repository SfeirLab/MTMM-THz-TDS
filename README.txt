To try different structures in the main file, simply modify the following variables using the data in the Data folder.
You can also add your own data to this folder if needed.

SampleName – Specifies the sample to be used for refractive index calculation. This is a JSON file that contains the settings and sample structure.

Reference – Loads the experimental time trace of the reference.

Sample – Loads the experimental time trace of the sample.
